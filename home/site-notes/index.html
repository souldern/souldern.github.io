Editing the Website

In order to be able to edit the site you do need a GitHub account. Once you have a GitHub user name, Contact the site maintainers and ask to be added to the Souldern GitHub Organisation which will give you write access to the site.

Licence

As noted in the public About page unless otherwise noted the text and images on this website are Copyright Souldern Parish Council, and distributed under a Creative Commons Attribution-ShareAlike 4.0 International License.

Note this means that any contributing editors agree to assign copyright to the Parish Council. Other licences would be possible but the CC licences are a standard and well tested licence for this kind of use. it is highly recommended that some licence is used, to avoid problems with future edits to the text and general distribution issues. Care should be taken that any photographs obtained from the web should be compatible with this. Often this is not a problem even if the original is available in a more restrictive form, for example the 100th anniversary picture of a steam train on the viaduct was marked as “All rights reserved” which would prevent use on this site, however I mailed the author requesting permission and got a reply back the same day, a copy of which is in the Souldern Village gmail for the record. Similarly permission was requested and granted for the image of the pond in the site banner, again forwarded to the gmail account.

Editing the Calendar

Access to the calendar is handled separately to access to the website, anyone with Owner access to the calendar can “share” the calendar with another user via email from the “settings” page. Note as well as being available on the site if you use the button under the calendar the calendar may be imported into most calendar apps, eg on mobile phones, etc.

Photographs and Slideshows

The original version of the village hall picture was 10MB in size, that That is: one image doubled the file usage of the whole site at that time!

After uploading to the Google photo sharing site with the account setting “high” resolution giving unlimited storage space, the file size was reduced to 4MB.

Both of these versions are still way beyond what is needed for a web site and are not actually the versions a reader sees, as the google server serves much smaller images depending on the requested size, In this case the browser reports

https://sites.google.com/site/souldernuk/_/rsrc/1443475151045/
    village-hall/1114.JPG?height=213&width=320
20.01 KB (20,490 bytes)

So the file actually seen by the reader of the site is 0.02MB in size, so any theoretical loss of quality in the photo site storing a 4MB version rather than the original 10MB is dwarfed by the file reduction needed to make a web page display quickly. More importantly the 4MB version accounts for 0% of the site file quota.

However it is still recommended that the site site image upload is used initially rather than multiple people using the administrative Google Village account.

An alternative would be that individual editors upload photos to their own photo sharing accounts. This would have similar effect of not using up the site file quota, but would potentially fragment the site content making it harder to be sure that it was all under the village control as editors come and go.

Any images that are taken off the site will be re-added to the public gallery owned by the Souldern Village account.
https://picasaweb.google.com/111338960776702858271/SouldernWebsite
from there individual images can be re-added back to the site using the normal "insert image" dialog but using the web address of the picture rather than file upload.

Contact Form

As a matter of policy I have avoided putting most emails on the website as plain text. The main exception being emails to the souldern.org domain (if these get affected by spam mail we can discard them and allocate new addresses without affecting people's main email accounts).

As well as the general “Contact” link under the home page, several pages such as Village hall booking, Ann's Book banner in the history page and the Community market page use a link to the contact form as an alternative to direct email to the relevant person. All messages sent by the form are recorded in a spreadsheet in the Souldern Village Drive account, and that spreadsheet then sends an email to souldern.website [at] gmail.com email account. That gmail account is in turn set up to forward currently to David Carlisle and Nick Oakhill's email, but other email addresses could be added. For simplicity just one email form is used across the site although if needed individual forms could be set up to mail direct to different individuals.

The form is actually backed up by a spreadsheet stored in the Souldern Village Google Drive account. All responses are entered into that spreadsheet. A script on the spreadsheet then forwards the response to souldern.website [at] gmail.com.
